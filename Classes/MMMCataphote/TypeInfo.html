TypeInfo

  • unknown

    Used when we cannot figure out the contents of a decodable, like when it’s empty or has unsupported features.

  • primitive(_:)

    Simple type like Int, Double or String.

  • object(_:)

    Another Decodable struct or class.

  • array(_:)
  • dictionary(_:_:)
  • optional(_:)

    An optional wrapping the given type.

  • objects()

    Objects used in the description of this type (without recursing into fields of these objects). This to be able to see objects through optionals or arrays.
